Prática 1


É hora de praticar!

Você pode fazer as atividades aqui e conferir as respostas. Se preferir faça direto no gerenciador do banco de dados!


1. Crie um banco de dados chamado 'musicas'

CREATE DATABASE musicas



2. Crie uma tabela de 'Generos' com a seguinte estrutura:

id (chave primária, inteiro)

nome_genero (texto: 10 caracteres)

CREATE TABLE generos ( id INT NOT NULL AUTO_INCREMENT, nome_genero VARCHAR(10), PRIMARY KEY (id));



3. Crie a tabela de 'Artistas' com a seguinte estrutura:

id (chave primária, inteiro)

nome_artista (texto, 30 caracteres)

data_nascimento (data)

avaliacao (número decimal 6,2)

CREATE TABLE artistas ( id INT NOT NULL AUTO_INCREMENT, nome_artista VARCHAR(30), data_nascimento DATE, avaliacao DECIMAL(6,2), PRIMARY KEY (id));



4. Altere a estrutura da tabela 'Artistas', adicionando o campo 'genero'​ (chave estrangeira).

ALTER TABLE artistas ADD genero INT;



5. Altere a estrutura da tabela 'Artistas', adicionando o campo 'pais_origem'​ (texto 10 caracteres).

ALTER TABLE artistas ADD pais_origem VARCHAR(10);



6. Altere a estrutura da tabela 'Artistas', modificando o campo '​pais_origem (texto com 20 caracteres).

ALTER TABLE artistas MODIFY pais_origem VARCHAR(20);



7. Altere a estrutura da tabela 'Artistas', excluindo o campo​ 'data_nascimento'

ALTER TABLE artistas DROP data_nascimento;



8. Mostre todas as tabelas da base de dados 'musicas'.

SHOW tables;



9. Exclua a base de dados 'musicas'.

DROP DATABASE musicas;
